6. Mô hình lưu trữ dữ liệu

Hình 3.6: Mô hình lưu trữ dữ liệu Browser Storage
BROWSER STORAGE
LOCAL STORAGE (Dữ liệu lưu vĩnh viễn)
'products' → JSON Array (Danh sách sản phẩm)
'cart' → JSON Array (Giỏ hàng)
'users' → JSON Array (Danh sách người dùng)
'orders' → JSON Array (Danh sách đơn hàng)
'reviews' → JSON Array (Đánh giá sản phẩm)
'contacts' → JSON Array (Tin nhắn liên hệ)
SESSION STORAGE (Dữ liệu phiên làm việc)
'currentUser' → JSON Object (Người dùng đang đăng nhập)
'lastOrderId' → Number (Mã đơn hàng vừa tạo)

9. Sơ đồ liên kết giữa các trang

Hình 3.9: Sơ đồ liên kết và điều hướng giữa các trang
Từ trang Liên kết đến Điều kiện
index.html → sanpham.html, chitiet.html, giohang.html, login.html, lienhe.html, danhgia.html -
sanpham.html → chitiet.html (xem chi tiết), giohang.html (thêm giỏ) -
chitiet.html → giohang.html (thêm vào giỏ), login.html (chưa đăng nhập) Cần đăng nhập để thêm giỏ
giohang.html → checkout.html (đặt hàng), sanpham.html (tiếp tục mua) Giỏ hàng không rỗng
checkout.html → order-success.html (thành công), giohang.html (quay lại) Điền đầy đủ thông tin
login.html → account.html (sau đăng nhập), index.html (trang trước) Đăng nhập/Đăng ký thành công
account.html → login.html (đăng xuất), chitiet.html (xem đơn hàng) Đã đăng nhập
admin-login.html → admin.html (đăng nhập thành công) Tài khoản Admin
admin.html → admin-login.html (đăng xuất) Đã đăng nhập Admin

12. Cấu trúc dữ liệu chi tiết

12.1. Cấu trúc dữ liệu Sản phẩm (Product)

Tên trường Kiểu dữ liệu Mô tả
id Number Mã sản phẩm (khóa chính)
name String Tên sản phẩm
brand String Hãng sản xuất (Apple, Samsung, Xiaomi...)
price Number Giá bán (VNĐ)
oldPrice Number Giá gốc trước khuyến mãi
discount Number Phần trăm giảm giá
image String Đường dẫn hình ảnh sản phẩm
ram Number Dung lượng RAM (GB)
storage Number Bộ nhớ trong (GB)
rating Number Đánh giá sao (1-5)
specs Object Thông số kỹ thuật chi tiết
description String Mô tả sản phẩm

12.2. Cấu trúc dữ liệu Đơn hàng (Order)

Tên trường Kiểu dữ liệu Mô tả
id Number Mã đơn hàng (khóa chính)
userId Number Mã người dùng (khóa ngoại)
orderNumber String Số đơn hàng (DHxxxxxx)
customerName String Tên người nhận hàng
customerPhone String Số điện thoại người nhận
address String Địa chỉ giao hàng
items Array Danh sách sản phẩm trong đơn
total Number Tổng tiền đơn hàng
paymentMethod String Phương thức thanh toán
status String Trạng thái đơn hàng
createdAt String Thời gian tạo đơn

12.3. Cấu trúc dữ liệu Người dùng (User)

Tên trường Kiểu dữ liệu Mô tả
id Number Mã người dùng (khóa chính)
name String Họ và tên
email String Địa chỉ email (unique)
phone String Số điện thoại
password String Mật khẩu
createdAt String Ngày tạo tài khoản
